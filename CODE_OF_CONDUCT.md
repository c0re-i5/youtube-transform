# Code of Conduct

This project follows the
[Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

## In short

- Be respectful. Disagree about code, not about people.
- No harassment, doxxing, or personal attacks in any project space (issues,
  PRs, discussions, commits).
- Assume good faith. Ask clarifying questions before escalating.
- Project maintainers may close, edit, or remove contributions that violate
  these norms.

## Reporting

If you experience or witness behavior that violates this code, open a
confidential discussion with the maintainer via GitHub, or open an issue
with the `coc` label if public discussion is appropriate.
